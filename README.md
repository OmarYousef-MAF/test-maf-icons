## Installation

### Gradle (Android)

Add to your `build.gradle.kts`:

```kotlin
dependencies {
    implementation("com.majidalfuttaim.icons:maf-icons:0.0.15")
}
```

### Maven Repository

Add GitHub Packages repository:

```kotlin
repositories {
    maven {
        url = uri("https://maven.pkg.github.com/majidalfuttaim/maf-icons-mobile")
        credentials {
            username = project.findProperty("gpr.user") as String?
            password = project.findProperty("gpr.token") as String?
        }
    }
}
```

## Usage

Access SVG files as resources:

```kotlin
// Android
val inputStream = resources.openRawResource(R.raw.home_line_24)

// Kotlin Multiplatform
val svg = this::class.java.getResourceAsStream("/icons/home_line_24.svg")
```
